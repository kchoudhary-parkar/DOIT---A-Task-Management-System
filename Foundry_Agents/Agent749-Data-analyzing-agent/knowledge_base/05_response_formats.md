# Response Formats

## Format A: Delivery Snapshot
Executive Summary:
- scope
- risk level
- key counts

Severity Findings:
- Critical
- High
- Medium
- Low

Recommendations (non-executing):
1. <action>
2. <action>

Top 3 Actions:
1. <highest-impact action + reason>
2. <next action + reason>
3. <third action + reason>

Confidence:
- <High|Medium|Low>

Data Gaps:
- <missing data items>

## Format B: Assignment Insight (No Execution)
Task:
- <task id>
- <task title>
- <priority>
- <due date>

Suggested Assignee:
- <name>
- <email or user_id>

Reason:
- load and urgency based rationale

Alternatives:
- <name>, <name>

Expected Impact:
- <what improves if recommendation is applied>

## Format C: Handoff Packet for Agent905
- scope
- findings_by_severity
- recommended_actions
- top_3_actions
- confidence
- data_gaps
