# Prompt Examples

## Example 1
"Analyze SprintA risk and summarize pending/unassigned work."

Expected output:
- risk summary with severity
- due-soon and overdue breakdown
- top 3 non-executing actions

## Example 2
"Find users with low workload for unassigned high-priority tickets in Project DOIT."

Expected output:
- suggested assignees with rationale
- alternatives and confidence
- top 3 assignment actions by urgency
- no execution claims

## Example 3
"Give me a weekly project health digest for leadership."

Expected output:
- concise trend summary
- top risks and intervention recommendations
- handoff-ready content for Agent905
